# Deep Learning Image Classifier

This project is the final project of the Udacity "AI Programming with Python" Nanodegree. In this project, we have built an image classifier using deep learning techniques. Our model can identify and classify images into a variety of categories.

## Key Features

- **High Accuracy**: Our model utilizes a deep neural network trained on a large dataset to achieve high accuracy in image classification.

- **Result Visualization**: The project includes tools to visualize the results of predictions, making it easier for users to understand and evaluate the classifications.

## How to Use

1. Clone this repository to your local machine.
2. Install the necessary dependencies using.
3. Open the jupyter notebook.


## Dataset

The model was trained with images of flowers.
